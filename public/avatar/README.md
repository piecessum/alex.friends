# Ассеты аватара

Используются такие файлы (имена — как загрузил Алексей):

| Файл            | Что это                       | Когда показывается              |
| --------------- | ----------------------------- | ------------------------------- |
| `day.PNG`       | статика **в очках**           | светлая тема (покой)            |
| `night.PNG`     | статика **без очков**         | тёмная тема (покой)            |
| `day-night.gif` | **снимает** очки (день→ночь)  | при переключении свет → тьма   |
| `night-day.gif` | **надевает** очки (ночь→день) | при переключении тьма → свет   |

Пути и длительность анимации заданы в `components/avatar-toggle.tsx`
(`ASSETS` и `ANIM_MS`).

## Что было сделано с гифками

- Убрано бесконечное зацикливание: гифки проигрываются **один раз** и замирают
  на последнем кадре (он совпадает со статичным фото).
- Оптимизированы: уменьшены до 320px, проредены до 36 кадров и ускорены до
  ~1.5 с. Размеры: day-night 16 МБ → ~2.1 МБ, night-day 3.7 МБ → ~1.9 МБ.
- Оригиналы лежат в `_originals/` (в деплой не попадают, см. `.gitignore`).
  Параметры пережатия можно повторить из истории команд (PIL: TARGET_LONG=320,
  TARGET_FRAMES=36, TARGET_TOTAL=1500).

## На заметку

- Скорость анимации регулируется парой: длина гифок (TARGET_TOTAL при пережатии)
  и `ANIM_MS` в `components/avatar-toggle.tsx` — держи их согласованными.
- Если когда-нибудь появится ffmpeg, перевод в `.webm`/`.mp4` даст ещё меньший
  размер при лучшем качестве (но потребует заменить `<img>` на `<video>`).
